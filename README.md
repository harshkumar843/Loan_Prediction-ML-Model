# Loan_Prediction-ML-Model
Trained an ML model on Support Vector Machine to detect the status of loan approval or rejection for multiple customers. 
The dataset contains 614 values. 
The parameters on which decision has been made are loan amount, no. of dependents, employment, property area, marital status, credit history, etc.
Achieved an accuracy score of 80% and 83% on training and test data respectively, hence provides proper fitting of curve.
